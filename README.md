Personal Portfolio Website
This project is a responsive personal portfolio website developed using HTML5 and CSS3. The website showcases personal information, skills, and a contact form, and it is designed to adapt to different screen sizes using media queries. The website is hosted online using GitHub Pages.

Overview
The purpose of this project is to design and develop a simple, clean, and responsive portfolio website that represents personal skills and basic web development knowledge. It follows standard web design practices and is suitable for academic submission as well as personal use.

Technologies Used
HTML5 – For structuring the content of the website
CSS3 – For styling, layout, and responsiveness
Flexbox – For layout alignment
Media Queries – For responsive design
GitHub Pages – For hosting the website online

Responsive Design
The website is fully responsive and works on:
Desktop screens
Tablets
Mobile devices
This is achieved using CSS Flexbox and media queries to adjust layout and alignment based on screen width.
