## Personal Portfolio Website
A responsive personal portfolio website built using **HTML5** and **CSS3**, designed to showcase personal information, skills, and a contact form. The website follows modern web design principles and is hosted using **GitHub Pages**.

---

## Project Overview
This project focuses on creating a clean and responsive portfolio website suitable for academic submission and personal branding. The website adapts seamlessly across different screen sizes using CSS **Flexbox** and **media queries**.

---

## Technologies Used
- **HTML5** – Website structure  
- **CSS3** – Styling and layout  
- **Flexbox** – Responsive alignment  
- **Media Queries** – Mobile responsiveness  
- **GitHub Pages** – Website hosting  

---

## Website Sections
- **Home** – Introduction  
- **About** – Personal background
- **Hobbies** – Interest and Favourties  
- **Skills** – Technical and professional skills  


---

## Responsive Design
The website is fully responsive and optimized for:
- Desktop
- Tablet
- Mobile devices
